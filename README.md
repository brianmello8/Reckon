# Reckon

Read-only observability and anomaly detection for AI/LLM spend — polls provider APIs (Anthropic, OpenAI, GitHub Copilot) and surfaces per-developer costs, trends, and anomalies via Slack and Linear.

## Getting started

```bash
pnpm install
cp .env.example .env.local   # fill in real values
pnpm dev
```
